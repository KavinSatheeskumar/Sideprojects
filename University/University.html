Basic Neural Network

This side project is an implementation of multilayer perceptron with backpropagation learning. It was made from scratch without the use of external libraries like NumPy or TensorFlow and it is capable of training a neural network to do various simple tasks like evaluating the exclusiveor function or calculating the square of the sine of a number. The code for this project is avalible on the GitHub Repository for this website.
